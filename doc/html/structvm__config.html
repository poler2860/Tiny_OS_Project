Detailed Description

Virtual machine configuration.

The configuration of the virtual machine consists of the following information:

  • The boot function to execute on each core of the simulated machine, stored in bootfunc.
  • The number of CPU cores of this VM, stored in cores
  • The number of serial devices of this VM, stored in serialno
  • For each serial device, two file descriptors must be provided: the keyboard (serial_in) file descriptor will be read from and the console (serial_out) file descriptor will be written to. These file descriptors should correspond to some pipe-like Linux stream (e.g., pipe, FIFO or socket).

◆ serialno

uint vm_config::serialno

The number of serial ports connected to terminals that the computer will support.

The number of serial devices should be between 0 and MAX_TERMINALS.

The terminals can be accessed via pipes, which must already exist. The file descriptors for these pipes are passed in the kbd and con arrays of this structure.
