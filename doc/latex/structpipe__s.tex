\doxysubsection{Detailed Description}
A pair of file ids, describing a pipe. 

This structure is initialized by the {\ttfamily \mbox{\hyperlink{group__syscalls_gab6355ce54e047c31538ed5ed9108b5b3}{Pipe()}}} system call with two file descriptors, for the read and write ends of the pipe respectively. Writing bytes to the write end using {\ttfamily \mbox{\hyperlink{group__syscalls_gaf046f003fde24f79fb395c250137856c}{Write()}}} will make them available at the read end, unsing {\ttfamily \mbox{\hyperlink{group__syscalls_ga3e9dc545a789eb45b2d356eabbac3ee3}{Read()}}}.
